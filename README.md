# Shadow Jump

## Descrição

Shadow Jump é um jogo 2D de plataforma desenvolvido utilizando a Unity Engine e a linguagem C#.

O jogador deve atravessar obstáculos, evitar inimigos e avançar pelas fases do jogo utilizando movimentação lateral e saltos.

## Mecânicas

* Movimentação lateral
* Sistema de salto
* Inimigos
* Coleta de cristais
* Progressão entre fases
* Sistema de vitória ao final da fase

## Controles

* A ou Seta Esquerda: mover para esquerda
* D ou Seta Direita: mover para direita
* Espaço: pular

## Tecnologias Utilizadas

* Unity Engine
* C#
* Visual Studio Code
* GitHub

## Estrutura do Projeto

O projeto foi organizado utilizando cenas, scripts e assets separados para facilitar a manutenção e desenvolvimento.

## Como Executar

1. Baixe ou clone o repositório.
2. Abra o projeto na Unity.
3. Carregue a cena inicial do jogo.
4. Pressione Play para iniciar a execução.
